# Agent Skills

A collection of agent skills that extend capabilities across planning, development, and tooling.

## development

These skills help you write, refactor, and fix code.

- **haskell-beam-postgres** — this helps the agent to write better beam queries in haskell. Prequisite: any web search mcp. 

  ```
  npx skills@latest add rohanafterdark/hypercredit-exa/haskell-beam-postgres
  ```

- **process-tracker** - this skill helps the agent to write better process trackers.

  ```
  npx skills@latest add rohanafterdark/hypercredit-exa/process-tracker
  ```

- **clean-code** - skill to analyze the codebase or part of the codebase to follow clean code principles

  ```
  npx skills@latest add rohanafterdark/hypercredit-exa/clean-code
  ```
